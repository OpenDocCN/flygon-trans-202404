# 建模

预测建模是创建模型以预测结果的过程。如果结果是分类的，则称为[分类](classification.htm)，如果结果是数值的，则称为[回归](regression.htm)。描述性建模或[聚类](clustering.htm)是将观察结果分配到簇中，使得同一簇中的观察结果相似。最后，[关联规则](association_rules.htm)可以在观察结果中找到有趣的关联。
